# Agmentic Agent Handshake

Static GitHub Pages demo for a live consumer-agent and retailer-agent interaction.

It shows:

- proximity handshake between consumer and retailer agents
- retailer menu and offer transfer
- consumer counter-request
- accepted negotiation terms
- the same stream in agent protocol language and plain English

The page can connect to a compatible FastAPI backend at `/api/*`. On GitHub Pages, where no backend is available, it automatically runs a browser-based live simulation with the same event schema.
